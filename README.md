# Toorakarn AI Assistant (น้องธุรการ) 💜⚡

**น้องธุรการ** คือ AI Chatbot อัจฉริยะที่พัฒนาขึ้นเพื่อเป็นผู้ช่วยส่วนตัวของพนักงาน **การไฟฟ้าส่วนภูมิภาค (กฟภ. / PEA)** ในการค้นหาข้อมูลสวัสดิการ ระเบียบข้อบังคับ และขั้นตอนการดำเนินงานด้านธุรการต่างๆ ผ่านทาง LINE

## 🌟 ความสามารถหลัก (Core Capabilities)

1. **ตอบคำถามสวัสดิการแม่นยำ (RAG Integration):**
   - ค้นหาคำตอบจากฐานข้อมูลระเบียบสวัสดิการของ กฟภ. โดยตรง
   - รองรับการถาม-ตอบเรื่อง ค่ารักษาพยาบาล, ค่าเล่าเรียนบุตร, เงินช่วยพิเศษ, และสิทธิการลาต่างๆ
   - **Strict Grounding:** ตอบเฉพาะข้อมูลที่มีในฐานข้อมูลเท่านั้น เพื่อป้องกันการให้ข้อมูลผิดพลาด

2. **รองรับข้อความเสียง (Voice-to-Query):**
   - พี่ๆ สามารถส่งข้อความเสียงถามน้องธุรการได้โดยไม่ต้องพิมพ์
   - ระบบจะถอดความและค้นหาคำตอบให้ทันที เหมาะสำหรับเวลาเร่งด่วน

3. **ค้นหาข้อมูลแบบ Semantic Search:**
   - ใช้เทคโนโลยี Vector Search (Cloudflare Vectorize) ทำให้เข้าใจความหมายของคำถาม แม้จะใช้คำเรียกที่ไม่ตรงกับระเบียบเป๊ะๆ

4. **บุคลิกภาพที่เป็นมิตร (Persona-driven):**
   - ตอบกลับด้วยภาษาที่เข้าใจง่าย อบอุ่น และเป็นกันเอง (น้องธุรการ 💜⚡)
   - จัดรูปแบบข้อความให้อ่านง่ายบนมือถือ (Chunking & Emojis)

## 🛠 เทคโนโลยีที่ใช้
- **Framework:** Cloudflare Workers (TypeScript)
- **AI Model:** Google Gemini 3.1 Flash Lite (Reasoning & Audio Processing)
- **Database:** Cloudflare Vectorize & KV Storage
- **Interface:** LINE Messaging API
